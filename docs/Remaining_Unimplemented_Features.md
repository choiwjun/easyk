## 1. P2 우선순위 (낮음) - 미구현 기능

### 1.1 서류 템플릿 다운로드

**UserFlow 정의**:
- 사용자가 정부 지원 신청 시 필요한 서류 템플릿 다운로드
- 예시: 이력서 양식, 신청서, 자기소개서 템플릿

**현재 상태**: ❌ **미구현**

**구현 필요 사항**:
- 백엔드 API:
  - `GET /api/supports/{id}/templates` - 지원 프로그램별 템플릿 목록
  - `GET /api/supports/{id}/templates/{template_id}/download` - 템플릿 다운로드
  
- 백엔드 모델:
  - `DocumentTemplate` 모델 생성
  - `document_templates` 테이블 생성
  
- 프론트엔드 UI:
  - 지원 프로그램 상세 페이지에 "서류 템플릿 다운로드" 버튼
  - 템플릿 목록 모달
  - 다운로드 진행 상태 표시

**구현 예상 시간**: 1주

**개선 필요 사항**:

### 2.3 파일 업로드 기능

**UserFlow 정의**:
- 사용자가 이력서, 사진 등의 파일 업로드
- 지원서 작성 시 파일 첨부

**현재 상태**: ❌ **미구현** (현재는 URL만 지원)

**구현 필요 사항**:
- 백엔드 API:
  - `POST /api/uploads` - 파일 업로드 엔드포인트
  - 파일 형식, 크기 제한
  - S3 또는 로컬 스토리지 연동
  
- 백엔드 모델:
  - `Upload` 모델 (파일 정보 저장)
  - `uploads` 테이블
  
- 프론트엔드 UI:
  - 파일 업로드 컴포넌트
  - 드래그 앤 드롭 지원
  - 파일 프리뷰

**구현 예상 시간**: 1주

### 2.4 채팅 기능

**UserFlow 정의**: 상담 진행 시 실시간 메시지 교환
