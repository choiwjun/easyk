## 1. FEAT-1: 법률 상담 신청 & 매칭

### 1.1 외국인 사용자 흐름

| 단계 | UserFlow 정의 | 실제 구현 | 상태 |
|------|---------------|-----------|------|
| 로그인/회원가입 | Login/Signup 페이지 | ✅ | 완료 |
| 프로필 작성 | `/profile` 페이지 | ✅ | 완료 |
| 상담 신청 | `/consultations/new` 페이지 | ✅ | 완료 |
| 상담 유형 선택 | 5가지 유형 선택 (visa, labor, contract, business, other) | ✅ | 완료 |
| 상담 내용 입력 | 폼 입력 및 검증 | ✅ | 완료 |
| 전문가 선택/매칭 | 자동 매칭 | ⚠️ | 부분 구현 |
| 상담 목록 조회 | `/consultations` 페이지 | ✅ | 완료 |
| 상담 상세 보기 | `/consultations/[id]` 페이지 | ❌ | **누락** |
| 예약 화면 | 결제 전 예약 확인 | ⚠️ | 결제 페이지로 대체 |
| 결제 | `/consultations/[id]/payment` | ✅ | 완료 |
| 결제 완료/실패 | `/consultations/[id]/payment/success`, `/consultations/[id]/payment/fail` | ✅ | 완료 |
| 상담 진행 | 상담 예정일 대기 | ⚠️ | 상태만 표시 |
| 상담 완료 | status: 'completed' | ✅ | 완료 |
| 후기 작성 | `/consultations/[id]/review` 페이지 | ✅ | 완료 |

**누락된 기능**:
- `/consultations/[id]` - 상담 상세 페이지 (현재 결제 페이지로 바로 이동)
- 전문가 선택 UI (현재 자동 매칭만)
- 상담 취소 기능

## 전체 구현율

### 완료 구현 (✅)

1. **공통 인증**:
   - 로그인/회원가입 페이지
   - 프로필 설정 페이지
   - 토큰 인증

2. **법률 상담 (외국인)**:
   - 상담 신청 폼
   - 상담 목록
   - 결제 페이지 (토스페이먼츠)
   - 결제 성공/실패 페이지
   - 후기 작성 페이지
   - 상태 표시 (requested, matched, scheduled, completed, cancelled)

3. **일자리 찾기 (외국인)**:
   - 일자리 목록
   - 검색/필터링 (키워드, 지역, 근로형태)
   - 일자리 상세 페이지
   - 샘플 데이터 데모 모드

4. **정부 지원 정보 (외국인)**:
   - 지원 프로그램 목록
   - 카테고리 필터링
   - 키워드 검색
   - 프로그램 상세 페이지
   - 외부 링크 연결
   - 담당 기관 정보
   - 샘플 데이터 데모 모드

### 부분 구현 (⚠️)

1. **법률 상담**:
   - 자동 매칭만 구현 (전문가 선택 UI 미구현)
   - 이메일 알림만 구현

2. **일자리 찾기**:
   - 지원서 작성 UI 확인 필요
   - 지원 내역 확인 필요

3. **정부 지원**:
   - 외부 링크만 연결 (자격 확인 폼 미구현)

### 누락 구현 (❌)

1. **법률 상담 - 전문가용**:
   - `/consultants` - 전문가 대시보드
   - `/consultants/incoming` - 진입 상담 목록
   - 전문가용 상담 상세 페이지
   - 전문가 수락/거절 UI
   - 상담 취소 기능

2. **일자리 - 관리자용**:
   - `/admin/jobs` - 관리자 대시보드
   - 공고 등록/수정/삭제 UI
   - 지원자 목록 및 상세
   - 채용/거절 피드백 UI
   - 통계 대시보드
   - 관심 공고 저장 (찜하기)

3. **정부 지원**:
   - 자격 확인 폼
   - 서류 템플릿 다운로드
   - 신청 상태 추적

## 우선순위 권장 사항

### 높음 우선순위 (P0)

1. **상담 상세 페이지** (`/consultations/[id]`)
   - 상담 정보 상세 표시
   - 전문가 정보
   - 결제/예약/취소 버튼
   - 상태 이력

2. **전문가 대시보드** (`/consultants`)
   - 진입 상담 목록
   - 수락/거절 버튼
   - 상담 일정 관리

3. **지원서 작성 UI** (`/jobs/[id]/apply`)
   - 이력서, 자기소개
   - 추가 정보 입력
   - 지원 제출

### 중간 우선순위 (P1)

4. **관리자 대시보드** (`/admin/jobs`)
   - 공고 관리
   - 지원자 관리
   - 통계 대시보드

5. **자격 확인 폼** (`/supports/[id]/eligibility`)
   - 비자 종류 확인
   - 자격 결과 표시
   - 유사 프로그램 추천

### 낮음 우선순위 (P2)

6. **관심 공고 저장**
7. **지원 내역 페이지**
8. **신청 상태 추적**
9. **서류 템플릿**
