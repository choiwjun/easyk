## 회원가입

### 계정 생성 방법

1. **회원가입 페이지 접속**
   - 메인 화면에서 "회원가입" 버튼 클릭
   - 또는 URL 직접 접속: `https://easyk.com/signup`

2. **회원 유형 선택**
   - **외국인 회원**: 한국에 거주하거나 입국 예정인 외국인
   - **지원 기관**: 전문가, 상담 기관

3. **필수 정보 입력**
   - **이름**: 실명으로 입력 (성, 이름 따로)
   - **이메일**: 실제 사용 가능한 이메일 주소
   - **비밀번호**: 8자 이상, 영문/숫자/특수문자 포함
   - **이용약관**: 체크박스 필수 동의

4. **이메일 중복 확인**
   - 이메일 입력 시 자동으로 중복 확인
   - 이미 사용 중인 이메일은 경고 메시지 표시

5. **회원가입 완료**
   - "회원가입" 버튼 클릭
   - 자동으로 메인 페이지로 이동

### 비밀번호 보안 가이드

- ✅ 8자 이상 사용
- ✅ 영문 대소문자, 숫자, 특수문자 조합
- ❌ 생일, 전화번호 등 쉽게 유추 가능한 숫자 조합 금지
- ❌ 다른 사이트와 동일한 비밀번호 사용 금지

## 로그인

### 로그인 방법

1. **로그인 페이지 접속**
   - 메인 화면에서 "로그인" 버튼 클릭
   - 또는 URL 직접 접속: `https://easyk.com/login`

2. **정보 입력**
   - **이메일**: 가입 시 사용한 이메일
   - **비밀번호**: 설정한 비밀번호

3. **로그인 완료**
   - "로그인" 버튼 클릭
   - 역할에 따라 대시보드로 이동:
     - 외국인: 일자리 목록 (`/jobs`)
     - 전문가: 상담 목록 (`/consultations`)
     - 관리자: 관리자 대시보드 (`/admin/jobs`)

### 비밀번호 찾기

1. 로그인 페이지에서 "비밀번호 찾기" 클릭
2. 가입 시 입력한 이메일 주소 입력
3. 이메일로 임시 비밀번호 전송
4. 임시 비밀번호로 로그인 후 비밀번호 변경

## 프로필 설정

### 프로필 정보 수정

1. **프로필 페이지 접속**
   - 메인 화면에서 "프로필 수정" 클릭
   - 또는 URL: `https://easyk.com/profile`

2. **수정 가능한 정보**
   - **이름**: 성, 이름
   - **국적**: 국가명
   - **비자 유형**: E-1, D-2, F-2 등
   - **비자 만료일**: 여권/비자 만료일
   - **선호 언어**: 한국어(ko), 영어(en)
   - **연락처**: 전화번호
   - **거주 지역**: 예) 경기도 고양시 덕양구
   - **자기소개**: 자유롭게 작성

3. **저장**
   - "수정 완료" 버튼 클릭
   - 변경 사항이 즉시 반영됨

### 프로필 사진 변경

1. 프로필 페이지에서 현재 사진 클릭
2. 새로운 사진 선택 (최대 5MB)
3. 자동으로 업로드 및 변경 완료

## 법률 상담

### 상담 신청 방법

1. **상담 메뉴 접속**
   - 메인 화면에서 "법률 상담" 클릭
   - 또는 URL: `https://easyk.com/consultations`

2. **새 상담 신청**
   - "상담 신청하기" 버튼 클릭

3. **상담 정보 입력**
   - **상담 유형**: 비자, 노무, 계약, 비즈니스, 사회
   - **제목**: 상담 주제 요약 (예: 비자 연장 관련)
   - **내용**: 자세한 상담 내용 작성
   - **선호 날짜**: 상담 희망 날짜/시간

4. **전문가 선택**
   - 전문 분야별 필터링 가능
   - 전문가 프로필 확인 (평점, 경력, 요금)
   - 원하는 전문가 선택 또는 자동 매칭 요청

5. **신청 완료**
   - "신청하기" 버튼 클릭
   - 상담 대기 목록에 추가됨

### 상담 진행 단계

| 단계 | 상태 | 설명 |
|------|------|------|
| 1 | 신청됨 | 상담 신청 완료, 전문가 대기 중 |
| 2 | 매칭됨 | 전문가가 상담 수락 |
| 3 | 예약됨 | 상담 날짜/시간 확정 |
| 4 | 진행 중 | 상담 진행 중 |
| 5 | 완료됨 | 상담 완료 |
| 6 | 취소됨 | 취소된 상담 |

### 상담 예약 및 결제

1. **전문가 승인 대기**
   - 전문가가 신청을 검토
   - 일반적으로 1~2일 내 응답

2. **예약 확인**
   - 승인 시 알림 및 이메일 수신
   - 상담 날짜/시간 확정

3. **결제**
   - 상담료 결제 (토스페이먼츠)
   - 결제 방법: 신용카드, 계좌이체
   - 결제 완료 시 상담 예약 확정

4. **상담 진행**
   - 예약된 시간에 상담 진행
   - 온라인 또는 오프라인 상담 선택 가능

5. **후기 작성**
   - 상담 완료 후 후기 작성
   - 전문가 평점과 후기 내용 입력
   - 후기는 다른 사용자에게 참고 정보 제공

### 상담 취소

- 진행 전까지 언제든 취소 가능
- 마이페이지 > 상담 목록 > "취소" 버튼 클릭
- 취소 사유 선택 후 취소 완료
- 환불 정책에 따라 환불 처리

## 일자리 찾기

### 일자리 목록 조회

1. **일자리 메뉴 접속**
   - 메인 화면에서 "일자리 찾기" 클릭
   - 또는 URL: `https://easyk.com/jobs`

2. **검색 및 필터링**
   - **검색어**: 직종, 기업명, 지역 검색
   - **지역 필터**: 경기도 고양시, 서울특별시 등
   - **근로 형태**: 정규직, 계약직, 아르바이트
   - **언어 요건**: 한국어, 영어, 중국어 등

3. **일자리 상세 보기**
   - 관심 있는 공고 클릭
   - 상세 정보 확인:
     - 직무 내용
     - 자격 요건
     - 우대 사항
     - 급여 및 복지
     - 필요 언어 능력
     - 마감일

### 일자리 지원 방법

1. **지원 버튼 클릭**
   - 일자리 상세 페이지에서 "지원하기" 버튼 클릭

2. **지원서 작성**
   - **기본 정보**: 자동으로 프로필 정보 로드
   - **자기소개서**: 해당 일자리에 맞춰 작성
   - **경력/자격**: 관련 경력 및 자격증 입력
   - **선호 근무 시간**: 희망 근무 시간 선택

3. **제출**
   - "지원서 제출" 버튼 클릭
   - 지원 내역에 추가됨

4. **지원 결과 확인**
   - 마이페이지 > 지원 내역
   - 결과: 채용, 검토 중, 거절

### 지원 상태

| 상태 | 설명 | 다음 단계 |
|------|------|----------|
| 검토 중 | 기업에서 지원서 검토 중 | 대기 |
| 면접 대기 | 면접 일정 예정 | 면접 준비 |
| 최종 합격 | 채용 확정 | 입사 준비 |
| 불합격 | 채용 불합격 | 다른 공고 지원 |

## 정부 지원 정보

### 지원 프로그램 조회

1. **정부 지원 메뉴 접속**
   - 메인 화면에서 "정부 지원" 클릭
   - 또는 URL: `https://easyk.com/supports`

2. **카테고리 필터링**
   - **장려금**: 정착 지원금, 교육비 등
   - **교육**: 한국어 교육, 직업 훈련 등
   - **훈련**: 기술 훈련, 직무 교육 등
   - **비자**: 비자 연장, 체류 허가 등
   - **주거**: 임대주택, 주거 지원 등

3. **프로그램 상세 보기**
   - 관심 있는 프로그램 클릭
   - 상세 정보 확인:
     - 프로그램 설명
     - 자격 조건
     - 대상 비자 유형
     - 지원 내용 (금액, 기간 등)
     - 담당 기관 정보
     - 신청 기간

### 지원 프로그램 신청 방법

1. **자격 확인**
   - 내 비자 유형 확인
   - 해당 비자가 대상에 포함되는지 확인
   - 기타 자격 조건 확인

2. **외부 링크 접속**
   - "공식 신청 페이지" 버튼 클릭
   - 정부 기관 공식 웹사이트로 이동

3. **신청 절차**
   - 해당 기관의 신청 절차 따르기
   - 필요 서류 준비
   - 온라인 또는 오프라인 신청

4. **진행 상황 확인**
   - 신청 기관을 통해 확인
   - 보통 1~2주 내 결과 통지

## FAQ

### 회원가입/로그인

**Q: 회원가입 시 이메일 인증이 필요한가요?**
A: 현재 이메일 인증 없이 바로 회원가입이 가능합니다. 추후 보안 강화를 위해 도입 예정입니다.

**Q: 비밀번호를 잊어버렸어요.**
A: 로그인 페이지에서 "비밀번호 찾기"를 클릭하고 가입 시 입력한 이메일 주소를 입력하시면 임시 비밀번호가 발송됩니다.

**Q: 계정을 삭제하고 싶어요.**
A: 현재는 계정 삭제 기능을 지원하지 않습니다. 필요 시 고객센터에 문의해 주세요.

### 프로필

**Q: 프로필 정보는 언제까지 수정해야 하나요?**
A: 가입 후 7일 이내에 필수 정보(국적, 비자 유형)를 입력하는 것을 권장합니다. 이 정보는 맞춤형 서비스 제공에 사용됩니다.

**Q: 비자 정보가 변경되었어요.**
A: 프로필 페이지에서 비자 유형과 만료일을 수정할 수 있습니다. 변경 후 즉시 반영됩니다.

### 법률 상담

**Q: 상담 비용은 얼마인가요?**
A: 전문가별로 상이합니다. 평균 시간당 10~20만 원 정도입니다. 전문가 프로필에서 요금을 확인할 수 있습니다.

**Q: 상담이 거절되면 어떻게 하나요?**
A: 다른 전문가에게 다시 신청하거나, 자동 매칭 기능을 이용할 수 있습니다.

**Q: 상담 취소 시 환불은 어떻게 되나요?**
A: 상담 24시간 전까지 취소 시 100% 환불, 24시간 이내 취소 시 50% 환불됩니다.

**Q: 상담 후기는 언제 작성하나요?**
A: 상담 완료 후 7일 이내에 작성 가능합니다. 후기 작성 시 다른 사용자에게 도움이 됩니다.

### 일자리

**Q: 외국인도 지원할 수 있는 일자리인가요?**
A: 네, 모든 공고는 외국인 지원 가능한 공고입니다. 필요 언어 능력을 확인하고 지원해 주세요.

**Q: 지원서는 몇 건까지 작성할 수 있나요?**
A: 제한 없습니다. 여러 일자리에 동시 지원할 수 있습니다.

**Q: 지원 후 연락은 언제 오나요?**
A: 기업마다 상이하지만, 보통 지원 후 1주 내에 연락이 옵니다. 2주 이내 연락이 없을 경우 다른 공고를 검토해 보시길 권장합니다.

**Q: 면접은 어떤 언어로 진행되나요?**
A: 공고에 명시된 언어로 진행됩니다. 필요한 언어 능력이 있는지 확인 후 지원해 주세요.

### 정부 지원

**Q: 지원 프로그램 신청은 easyK에서 직접 하나요?**
A: 아니요, 각 정부 기관의 공식 웹사이트에서 신청해야 합니다. easyK는 정보 제공 및 안내 역할만 수행합니다.

**Q: 여러 지원 프로그램에 동시 신청 가능한가요?**
A: 대부분 가능합니다. 단, 중복 지원이 불가한 경우도 있으니 각 프로그램의 규정을 확인해 주세요.

**Q: 지원 결과는 언제 알 수 있나요?**
A: 기관마다 상이하지만, 보통 1~2주 내에 결과가 통지됩니다.

**Q: 지원 거절 시 재신청 가능한가요?**
